# Task Manager

A simple task management web application built with React and Node.js.

## Features

- User authentication
- Create, edit, and delete tasks
- Task filtering
- Responsive design

## Installation

```bash
git clone https://github.com/johndoe/task-manager.git
cd task-manager
npm install
```

## Run

```bash
npm run dev
```

Visit:

```
http://localhost:3000
```

## Tech Stack

- React
- Express
- MongoDB
- Tailwind CSS
